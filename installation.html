Using Conda¶

Note

CrowdDynamics has not yet been released as a Conda package. It will be released as soon as crowddynamics releases version 0.0.1.

From Source¶

Install Conda Package Manager¶

Install Conda package manager which . Miniconda is faster to install than the full Anaconda distribution which comes bundled with lots of other useful scientific packages, but are not required to run crowddynamics.

Download the Source Code¶

Note

Run all of the commands in the project root directory crowddynamics after you have downloaded the source code.

  1. Download the source code with git
git clone https://github.com/jaantollander/crowddynamics.git
  1. After the installation use conda to install a new environment and the dependensies to run crowddynamics
conda env create python3.5 -n crowd35 -f environment.yml
source activate crowd35
  1. Then install crowddynamics itself in editable mode using following pip command
pip install --editable .
  1. Now you can run crowddynamics from the commandline. Check out the available commands using
crowddynamics --help

Run Tests¶

Install dependencies to run the tests:

pip install -r requirements-tests.txt

Run test suite using pytest:

py.test
