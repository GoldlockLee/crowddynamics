crowddynamics.core.steering.navigation module¶

Navigation¶

Navigation aka path-planing algorithms computes target direction \(\mathbf{\hat{e}_0}\) for each agent. Target direction accounts for agents desire to move towards some direction.

Navigation algorithms can be combined. For examples static potential is combination of shortest path and obstacle handling.

crowddynamics.core.steering.obstacle_handling.obstacle_handling(dmap_obs, dir_map_obs, dir_map_targets, radius, strength)[source]¶

Function that merges two direction maps together. Let distance map from obstacles be \(\Phi(\mathbf{x})\) and \(\lambda(x)\) be any decreasing function \(\frac{\partial}{\partial\Phi}\lambda(x) < 0\) of distance from obstacles such that

\[\begin{split}\lambda(\Phi) &= \begin{cases} 1 & \Phi = 0 \\ 0 & \Phi > M > 0 \end{cases}\end{split}\]

Take weighted average between the direction from the obstacles and the direction from the target using \(p\) using the decreasing function defined above.

\[\begin{split}p &= \lambda(\Phi(\mathbf{x})) \\ \hat{\mathbf{e}}_{out} &= \mathcal{N}\big(p \hat{\mathbf{e}}_{obstacle} + (1 - p) \hat{\mathbf{e}}_{target}\big)\end{split}\]

Numerically this algorithm uses exponential function as \(\lambda\)

\[c^{\frac{x}{r}}\]

crowddynamics.core.steering.quickest_path.distance_map(mgrid: crowddynamics.core.steering.quickest_path.MeshGrid, targets: shapely.geometry.base.BaseGeometry, obstacles: typing.Union[shapely.geometry.base.BaseGeometry, NoneType]) → numpy.ndarray[source]¶

Distance map \(S(\mathbf{x})\) is obtained by solving Eikonal equation (Continuos shortest path problem) using fast marching Fast Marching Method (FMM).

\[\left \| \nabla S(\mathbf{x}) \right \| = \frac{1}{f(\mathbf{x})}, \quad \mathbf{x} \in \Omega\]

where

  • Function \(f : \bar{\Omega} \mapsto (0, +\infty)\) is the speed of travel at \(\mathbf{x}\)

Boundary conditions for the distance map

\[\begin{split}\begin{cases} S(\mathbf{x}) = 0, & \mathbf{x} \in \mathcal{E} \\ S(\mathbf{x}) \to \infty, & \mathbf{x} \in \mathcal{O} \end{cases}\end{split}\]

Initial values for speed

\[\begin{split}\begin{cases} f(\mathbf{x}) = 1, & \mathbf{x} \in \Omega \setminus \mathcal{O} \\ f(\mathbf{x}) \to 0, & \mathbf{x} \in \mathcal{O} \end{cases}\end{split}\]

crowddynamics.core.steering.quickest_path.travel_time_map()[source]¶

Dynamics potential takes into account the positions of the agents in the field. Equation

\[\left \| \nabla T(\mathbf{x}) \right \| = \frac{1}{f(\mathbf{x})}, \quad \mathbf{x} \in \Omega\]

and boundary conditions

\[\begin{split}f(\mathbf{x}) &= 1, \quad \mathbf{x} \in \Omega \setminus (\mathcal{O} \cup \mathcal{A}) \\ f(\mathbf{x}) &\leq 1, \quad \mathbf{x} \in \mathcal{A} \\ f(\mathbf{x}) &\to 0, \quad \mathbf{x} \in \mathcal{O}\end{split}\]
\[\frac{1}{f(\mathbf{x})} = 1 + \max \left( 0, c_{0} \left( 1 + c_{1} \frac{\mathbf{v} \cdot \nabla S(\mathbf{x})}{v_{0} \| \nabla S(\mathbf{x}) \|} \right) \right)\]
  • \(c_{0} > 0\) general impact strength
  • \(c_{1} > 0\) impact of the moving direction of an agent
