crowddynamics.core.integrator module¶

Integrator¶

Numerical solver for differential system consisting of translational and rotational motion which produces the movement for agents in crowddynamics

\[\begin{split}\begin{cases} m \frac{d^{2}}{d t^{2}} \mathbf{x}(t) = \mathbf{f}(t) + \boldsymbol{\xi}(t) \\ I \frac{d^{2}}{d t^{2}} \varphi(t) = M(t) + \eta(t) \end{cases}\end{split}\]

where

  • Total force \(\mathbf{f}(t) = \mathbf{f}^{adjusting} + \mathbf{f}^{agent-agent} + \mathbf{f}^{agent-obstacles}\)
  • Random fluctutation force \(\boldsymbol{\xi}(t)\)
  • Total torque \(M(t) = M^{adjusting} + M^{agent-agent} + M^{agent-obstacles}\)
  • Random fluctuation torque \(\eta(t)\)
crowddynamics.core.integrator.euler_integrator(agents, dt_min, dt_max)[source]¶

Differential system is integrated using numerical integration scheme using discrete adaptive timestep \(\Delta t\).

Acceleration on an agent

\[\begin{split}a_{k} &= \mathbf{f}_{k} / m \\ \mathbf{x}_{k+1} &= \mathbf{x}_{k} + \mathbf{v}_{k} \Delta t + \frac{1}{2} a_{k} \Delta t^{2} \\ \mathbf{v}_{k+1} &= \mathbf{v}_{k} + a_{k} \Delta t \\\end{split}\]

Angular acceleration

\[\begin{split}\alpha_{k} &= M_{k} / I \\ \varphi_{k+1} &= \varphi_{k} + \omega_{k} \Delta t + \frac{1}{2} \alpha_{k} \Delta t^{2} \\ \omega_{k+1} &= \omega_{k} + \alpha_{k} \Delta t \\\end{split}\]

Velocity verlet integrator algorithm

Translational motion

  1. \[\mathbf{v}_{k+1/2} = \mathbf{v}_{k} + \frac{1}{2} a_{k} \Delta t\]
  2. \[\mathbf{x}_{k+1} = \mathbf{x}_{k} + \mathbf{v}_{k+1/2} \Delta t\]
  3. \[a_{k+1} = \mathbf{f}_{k+1} / m\]

crowddynamics.core.quantities.density_classical(points, cell_size)[source]¶

Classical definition of density defined the density \(D\) as number of agents per unit of area.

\[D = \frac{N}{|A|}\quad \left[\mathrm{\frac{1}{m^{2}}}\right]\]

where

  • \(N\) is the number of agents
  • \(|A|\) is the size of the area

However the classical definition of density does not work as well when measuring density of crowd or granular media as it does for measuring density of fluids which have \(> 10^{18}\) particles per \(\mathrm{mm}^3\).

crowddynamics.core.quantities.density_voronoi(points, cell_size, definition=2)[source]¶

Density definition base on Voronoi-diagram. [Steffen2010]

Compute Voronoi-diagram for each position \(\mathbf{p}_i\) giving cells \(A_i\) to each agent \(i\). Area of Voronoi cell is denoted \(|A_i|\). Area \(A\) is the area inside which we measure the density.

Definition 1

\(S = \{i : \mathbf{q} \in A_i \wedge \mathbf{q} \in A\}\) is the set of agents whose Voronoi-diagram has points that belong to area \(A\).

\[\begin{split}p_i(\mathbf{x}) = \frac{1}{|A_i|} \begin{cases} 1 & \mathbf{x} \in A_i \\ 0 & \text{otherwise} \end{cases}\end{split}\]
\[p(\mathbf{x}) = \sum_{i \in S} p_i(\mathbf{x})\]
\[\begin{split}D_V &= \frac{1}{|A|} \int_A p(\mathbf{x}) d\mathbf{x} \\ &= \frac{1}{|A|} \int_A \sum_{i \in S} p_i(\mathbf{x}) d\mathbf{x} \\ &= \frac{1}{|A|} \sum_{i \in S} \int_A p_i(\mathbf{x}) d\mathbf{x} \\ &= \frac{1}{|A|} \sum_{i \in S} \frac{1}{|A_i|} \int_A \begin{cases} 1 & \mathbf{x} \in A_i \\ 0 & \text{otherwise} \end{cases} d\mathbf{x} \\ &= \frac{1}{|A|} \sum_{i \in S} \frac{|A \cap A_i|}{|A_i|}\end{split}\]
Definition 2

\(S = \{i : \mathbf{p}_i \in A\}\) is the set of agents and \(N = |S|\) the number of agents that is inside area \(A\).

\[D_{V^\prime} = \frac{N}{\sum_{i \in S} |A_i|}\]
Parameters:
  • points (numpy.ndarray) – Two dimensional points \(\mathbf{p}_{i \in \mathbb{N}}\)
  • cell_size (float) – Cell size of the meshgrid. Each cell represents an area \(A\) inside which density is measured.
  • definition (int) – Integer {1, 2} denoting which definition of density to use.
Returns:

Grid of values density values for each cell.
Return type:

numpy.ndarray
