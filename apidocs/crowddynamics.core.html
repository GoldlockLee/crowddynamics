crowddynamics.core.evacuation.agent_closer_to_exit[source]¶

Amount of positions (agents) closer to center of the exit.

  1. Denote \(i\) as indices of the positions \(\mathbf{x}\).
  2. Distance from narrow exit can be estimated
\[d_i = \| \mathbf{c} - \mathbf{x}_{i} \|\]
  1. By sorting the values \(d_i\) by its indices we obtain array
\[a_i = \underset{i \in P}{\operatorname{arg\,sort}}(d_i)\]

where

  • Values: Indices of the positions sorted by the distance from the exit
  • Indices: Number of positions closer to the exit
  1. By sorting the values \(a_i\) by it indices we obtain array
\[\lambda_i = \operatorname{arg\,sort} (a_i)\]

where

  • Values: Number of positions closer to the exit
  • Indices: Indices of the positions
Parameters:
  • c_door (numpy.ndarray) – Center of the exit \(\mathbf{c}\).
  • position (numpy.ndarray) – Positions \(\mathbf{x}\) of the agents.
Returns:

Array \(\lambda_i\)

numpy.ndarray

crowddynamics.core.evacuation.narrow_exit_capacity[source]¶

Estimation of the capacity of narrow exit.

Capacity estimation \(\beta\) of unidirectional flow through narrow bottleneck. Capacity of the bottleneck increases in stepwise manner.

Simple estimation

\[\beta_{simple} = c \left \lfloor \frac{d_{door}}{d_{agent}} \right \rfloor\]

More sophisticated estimation [Hoogendoorn2005a], [Seyfried2007a] when \(d_{door} \geq d_{agent}\)

\[\beta_{hoogen} = c \left \lfloor \frac{d_{door} - (d_{agent} - d_{layer})}{d_{layer}} \right \rfloor\]

We have relationship between simple and hoogendoorn’s estimation

\[\beta_{hoogen} = \beta_{simple}, \quad d_{layer} = d_{agent}\]
Parameters:
  • d_door (float) – Width of the door \(0\,\mathrm{m} < d_{door} < 3\,\mathrm{m}\).
  • d_agent (float) – Width of the agent \(d_{agent} > 0\,\mathrm{m}\).
  • d_layer (float, optional) – Width of layer \(d_{layer}\). Reference value: \(0.45\,\mathrm{m}\)
  • coeff (float) – Scaling coefficient \(c > 0\) for the estimation.
Returns:

Depending on the value of d_layer either simple or hoogen estimation is returned

  • None: Uses simple estimation \(\beta_{simple}\)
  • float: Uses Hoogendoorn’s estimation \(\beta_{hoogen}\)
Return type:

float

crowddynamics.core.integrator module¶

Integrator¶

Numerical solver for differential system consisting of translational and rotational motion which produces the movement for agents in crowddynamics

\[\begin{split}\begin{cases} m \frac{d^{2}}{d t^{2}} \mathbf{x}(t) = \mathbf{f}(t) + \boldsymbol{\xi}(t) \\ I \frac{d^{2}}{d t^{2}} \varphi(t) = M(t) + \eta(t) \end{cases}\end{split}\]

where

  • Total force \(\mathbf{f}(t) = \mathbf{f}^{adjusting} + \mathbf{f}^{agent-agent} + \mathbf{f}^{agent-obstacles}\)
  • Random fluctutation force \(\boldsymbol{\xi}(t)\)
  • Total torque \(M(t) = M^{adjusting} + M^{agent-agent} + M^{agent-obstacles}\)
  • Random fluctuation torque \(\eta(t)\)
crowddynamics.core.integrator.adaptive_timestep[source]¶

Timestep is selected from interval \([\Delta t_{min}, \Delta t_{max}]\) by bounding the maximum step size \(\Delta x\) an agent can take per iteration cycle, obtained from

\[\begin{split}\Delta x = c \Delta t_{max} \max_{i\in A} v_i^0 \\\end{split}\]

where

  • \(c > 0\) is scaling coefficient
  • \(v_i^0\) is agent’s target velocity
  • \(\max_{i\in A} v_i^0\) is the maximum of all target velocities

Timestep is then obtained from

\[\begin{split}\Delta t_{mid} &= \frac{\Delta x}{\max_{i \in A} v_i} \\ \Delta t &= \begin{cases} \Delta t_{min} & \Delta t_{mid} < \Delta t_{min} \\ \Delta t_{mid} & \\ \Delta t_{max} & \Delta t_{mid} > \Delta t_{max} \\ \end{cases}\end{split}\]

where

  • \(v_i\) is agent’s current velocity
Parameters:
  • dt_min – Minimum timestep \(\Delta x_{min}\) for adaptive integration.
  • dt_max – Maximum timestep \(\Delta x_{max}\) for adaptive integration.

crowddynamics.core.integrator.euler_integrator(agents, dt_min, dt_max)[source]¶

Differential system is integrated using numerical integration scheme using discrete adaptive timestep \(\Delta t\).

Acceleration on an agent

\[\begin{split}a_{k} &= \mathbf{f}_{k} / m \\ \mathbf{x}_{k+1} &= \mathbf{x}_{k} + \mathbf{v}_{k} \Delta t + \frac{1}{2} a_{k} \Delta t^{2} \\ \mathbf{v}_{k+1} &= \mathbf{v}_{k} + a_{k} \Delta t \\\end{split}\]

Angular acceleration

\[\begin{split}\alpha_{k} &= M_{k} / I \\ \varphi_{k+1} &= \varphi_{k} + \omega_{k} \Delta t + \frac{1}{2} \alpha_{k} \Delta t^{2} \\ \omega_{k+1} &= \omega_{k} + \alpha_{k} \Delta t \\\end{split}\]

  • dt_max (float) – Maximum timestep \(\Delta x_{max}\) for adaptive integration.

crowddynamics.core.integrator.velocity_verlet_integrator(agents, dt_min, dt_max)[source]¶

Velocity verlet integrator algorithm

Translational motion

  1. \[\mathbf{v}_{k+1/2} = \mathbf{v}_{k} + \frac{1}{2} a_{k} \Delta t\]
  2. \[\mathbf{x}_{k+1} = \mathbf{x}_{k} + \mathbf{v}_{k+1/2} \Delta t\]
  3. \[a_{k+1} = \mathbf{f}_{k+1} / m\]
  4. \[\mathbf{v}_{k+1} = \mathbf{v}_{k+1/2} + \frac{1}{2} a_{k+1} \Delta t\]
